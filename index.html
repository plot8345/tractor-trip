🔄 Import / Export & Sync

Google Drive Sync

Data is saved privately to your Google Drive. Auto-saves 3 seconds after every change. Access from any device.

Connect Google Drive from the top bar to enable cloud sync.
📤

Local Backup

Download all records as a backup file. Send via WhatsApp or Email to sync between devices.

📥

Local Restore (Merge)

Import backup file. New records are added, existing records stay safe. No duplicates.

📂 Tap to select .json backup file
🔁

Replace All Data

⚠️ Deletes current data and fully replaces with backup file. Use for full sync from another device.

📂 Tap to select .json backup file
🗑️

Clear All Data

Permanently delete all trips and payments from this device. Cannot be undone. Export first!

📖 How to Use Sync
☁️ Google Drive (Best): Connect once → auto-saves after every entry → open on any device, click Load from Drive
📱➡️💻 Manual Sync: Export on mobile → Send .json via WhatsApp → Open on laptop → Local Restore (Merge)
💡 Tip: Even with Drive connected, export a local backup every week for extra safety
